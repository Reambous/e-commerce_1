1. Tentukan Tujuan Proyek
Tipe project: Website toko online sederhana
Fitur utama: Homepage, Produk, Cart, Blog, Account 
Output: Website yang bisa dijalankan di localhost, lalu bisa di-hosting

2. Tentukan Stack yang Mudah untuk Pemula
Rekomendasi paling aman untuk semester 3:
Frontend: HTML, CSS, JavaScript
Framework opsional: Bootstrap / Tailwind
Backend: PHP atau Node.js
Database: MySQL
(Jika baru mulai, PHP + MySQL paling mudah.)

3. Buat Repository GitHub
Kamu: Buat repo
Teman: jadi collaborator
Buat branch utama: main
Buat branch kerja: frontend, backend, feature-<nama>
